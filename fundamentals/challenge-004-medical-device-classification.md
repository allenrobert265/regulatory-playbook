# 📖 Introduction

Not all medical devices present the same level of risk.

For example:

- A tongue depressor presents very little risk.
- An ECG machine presents moderate risk.
- A heart valve presents very high risk.

Because of these differences, regulatory authorities classify medical devices according to the level of risk they pose to patients and users.

Higher-risk devices require more evidence, testing, and regulatory review before they can be marketed.

---

# ❓ What is Medical Device Classification?

Medical Device Classification is the process of assigning a medical device to a risk category based on factors such as:

- Intended use
- Duration of contact with the patient
- Degree of invasiveness
- Whether the device is active or non-active
- Potential risks to the patient

The higher the risk, the stricter the regulatory requirements.

---

# 🌍 Why is Classification Important?

Classification determines:

- Regulatory pathway
- Documentation requirements
- Clinical evidence
- Risk management activities
- Testing requirements
- Level of regulatory review
- Post-market obligations

Incorrect classification can delay approval or result in regulatory non-compliance.

---

# 🇮🇳 India (CDSCO)

India classifies medical devices into four categories:

| Class | Risk Level |
|--------|------------|
| Class A | Low Risk |
| Class B | Low–Moderate Risk |
| Class C | Moderate–High Risk |
| Class D | High Risk |

---

# 🇺🇸 United States (FDA)

The FDA uses three classes:

| Class | Risk Level |
|--------|------------|
| Class I | Low Risk |
| Class II | Moderate Risk |
| Class III | High Risk |

The classification influences the type of submission required (e.g., 510(k), De Novo, or PMA).

---

# 🇪🇺 European Union (EU MDR)

The EU MDR also uses risk-based classification:

| Class | Risk Level |
|--------|------------|
| Class I | Low Risk |
| Class IIa | Low–Moderate Risk |
| Class IIb | Moderate–High Risk |
| Class III | High Risk |

Classification is determined using rules defined in Annex VIII of the EU MDR.

---

# 🔍 Factors That Influence Classification

When determining the classification of a device, regulators consider:

- Intended purpose
- Duration of use
- Invasiveness
- Contact with the body
- Whether the device is active
- Whether it delivers energy
- Whether it sustains or supports life
- Whether it contains software

---

# 📊 Examples

| Device | India | FDA | EU MDR |
|---------|--------|-----|---------|
| Tongue Depressor | Class A | Class I | Class I |
| Surgical Gloves | Class B | Class I or II* | Class I |
| Digital Thermometer | Class B | Class II | Class IIa |
| Pulse Oximeter | Class B | Class II | Class IIa |
| Infusion Pump | Class C | Class II | Class IIb |
| Pacemaker | Class D | Class III | Class III |

\*Classification can vary depending on the specific type and intended use.

---

# 💡 Key Takeaways

- Medical devices are classified according to risk.
- Different regions have different classification systems.
- Classification determines regulatory requirements.
- Intended use is one of the most important factors.
- Correct classification is one of the first responsibilities of a Regulatory Affairs professional.
